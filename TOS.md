# KabanosX Python Music

Bot muzyczny stworzony w 100% w Pythonie, który odtwarza muzykę z YouTube w Discordzie.  
Funkcje: odtwarzanie, kolejka, shuffle, skip, volume i więcej.  

## Instalacja
1. Sklonuj repozytorium  
2. Zainstaluj wymagane biblioteki: `pip install -r requirements.txt`
3. Skonfiguruj plik .env
4. Uruchom bota: `python main.py`

## Komendy
- `/play [utwór/link]` – dodaj utwór do kolejki  
- `/skip` – pomiń aktualny utwór  
- `/stop` – zatrzymaj odtwarzanie
